# Website Building Notes

## Tech Stack

- **React 18** + **Vite 6**
- **React Router v6** with HashRouter
- **markdown-it** for Markdown rendering
- **KaTeX** for math formulas

## Features

1. Pixel-style font (Press Start 2P)
2. Pink color theme (#FFB6C1)
3. Custom animated cursors (6 character themes)
4. Blog with Markdown + LaTeX support

Hosted on GitHub Pages at `https://oviedoLiao.github.io/`
